# Savery

A web application that helps college students discover affordable recipes, save their favorites, and create budget-friendly meal plans.

## Team Members

- James Jacob
- Melissa Rejuan

## Technologies

- HTML5
- CSS3
- JavaScript (ES6 Modules)
- Node.js
- Express.js
- MongoDB Atlas

## Project Objective

Savery helps college students eat well on a budget. Users can browse and filter recipes by cost, cuisine, protein content, and preparation time, save their favorite recipes, and build a weekly meal plan with an estimated total cost.

## How to Run

1. Clone the repository
2. Install dependencies:
npm install
3. Start the development server:
npm start
4. Open your browser and go to:
http://localhost:3000

## Pages

- **Recipe Discovery** — Browse and filter recipes
- **Create a Recipe** — Submit your own recipe
- **Plan Your Meals** — Build a weekly meal plan
- **Favorites** — View and manage your saved recipes
